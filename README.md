# nixblog.github.io

多电脑同步先安装
npm install hexo
npm install -g hexo-cli
npm install hexo-deployer-git

hexo new post "新建文章" #简写形式 hexo n "新建文章"
hexo clean #清除旧的public文件夹
hexo generate #生成静态文件 简写形式 hexo g
hexo deploy #发布到github上 简写形式 hexo d
